Frequently Asked Questions

Perimeter isn't showing a glow — what's wrong?

Make sure you have an upcoming Calendar event or Reminder within your urgency window. Perimeter only glows when an event is approaching. You can test it using the "Test Glow" button in Preferences.

My calendar events aren't showing up.

Perimeter needs Calendar access to work. Go to System Settings → Privacy & Security → Calendars and make sure Perimeter is enabled. Same for Reminders if you use those.

How do I restore my Pro purchase on a new Mac?

Open Perimeter → Preferences → About, and click "Restore Purchase." Make sure you're signed into the same Apple ID used for the original purchase.

Does Perimeter work with Google Calendar or Outlook?

Yes — any calendar account synced to the macOS Calendar app will work, including iCloud, Google, Outlook, and Exchange.

How does the free trial work?

Every download includes a free 7-day Pro trial — no credit card required. After the trial, the app reverts to the free tier. You can upgrade to Pro at any time from within the app.

Is Pro a subscription?

No. Pro is a one-time purchase of $9.99. Buy it once, own it forever.

What macOS version does Perimeter require?

Perimeter requires macOS 13 Ventura or later.
